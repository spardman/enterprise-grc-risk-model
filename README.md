# Enterprise GRC Risk Model: SaaS Data Security
**Project Lead:** Dwan Edwards, MBA

## 1. Scenario Definition
**Organization:** Global SaaS provider specializing in Human Capital Management (HCM).
**Assets at Risk:** 100,000+ customer PII records in a cloud environment.

## 2. Methodology
This project uses **Quantitative Risk Analysis** (Monte Carlo simulations) to move beyond "High/Medium/Low" labels. By running 10,000 simulations, I provide a dollar-value impact to inform budget and insurance strategy.

## 3. Findings
* **Primary Threat:** Ransomware Attack.
* **95% Value at Risk (VaR):** ~99,000.
